Website này sẽ giới thiệu một số thẻ lệnh cơ bản trong HTML

Một File HTML có đuôi là .html

Một cấu trúc HTML cơ bản:

< !DOCTYPE html>
  < html>

     < head>

     < /head>

      < body>

     < /body>

   < /html>

Các thẻ phổ biến trong< head> < /head>

  • < title>Tiêu đề cho tab website< /title>
  • < meta charset="utf8">: dùng để mã hóa trang web dưới dạng gì..
  • < meta name="Học code HTML" content="HTML LEARN"> : miêu tả trang
  • < meta name="keyword" content="HTML Learn HTML How to read HTML"> : tạo các keyword để người dùng tìm kiếm
  • < meta name="author" content="Nguyễn Văn A"> : tác giả trang web
  • < meta name="viewport" content="width=device-width,initial-scale=1.0"> : khỏi tạo độ rộng web tương thích với thiết bị
  • < link rel="stylesheet" href="web.css"> : đưa file css vào để apply cho page

< /ul>

AUDIO < audio> < /audio>

< audio • Attribute • controls>
  < source src="link." type="audio/mpeg">
  < source src="link dự phòng." type="audio/mpeg">
< /audio>

    ATTRIBUTE
  • controls (hiển thị điều khiển)
  • autoplay (tự động chạy)
  • autoplay muted (tự động chạy và tắt tiếng)
  • loop (lặp lại khi kết thúc)
